## Project Description

A modern CLI tool built in Rust that simplifies Kubernetes and cloud resource management through interactive workflows. Features include one-command deployments, automatic rollbacks, and cost estimation before applying changes.

## Technical Highlights

Built with Rust for performance and reliability, featuring:
- Interactive TUI for complex workflows
- Git-based configuration management
- Real-time resource monitoring
- Cost estimation and optimization suggestions
